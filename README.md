
# High Score Tracker
This is a program that includes a few casino games (poker, slots, blackjack), and will save your high score from each game. No real money is used, and your high score is just the amount of fake money you can make in the game. It is intended as a practice for managing files.

## How To Use
1. Run from main.py file, no downloads are required

## Project Features
- Multiple Casino-like games
- User Accounts (Just username and password)
- High Score Saving per Game
- Basic password encryption
- Admin functionality (Username is admin, password is 1234)
